发布房源时，您需按照《自如民宿品质审核标准》填写房源相关信息。房源信息应当真实，面积应当表明为建筑面积或者套内建筑面积；房源标题、房源描述以及周边介绍中不得出现新《广告法》禁用词汇，绝对化、夸张、无法准确定义和衡量的词汇。例如“最”、“第一”、“首家”、“独家”、“领先”、“王牌”、“完美”、“臻品”、“精装”、“高端”、“大约/左右”、“xx分钟到达”、“应有尽有”、“齐全”、“毗邻”、“紧挨着”、“零距离”等。

您承诺并保证您在自如民宿发布的房源及其中的设施能满足房客的居住用途，且具有安全性、适用性，否则因此导致房客或任意第三方人身、财产损失的，均由您承担责任。

具体退订政策设定应从下列三种方案中进行选择：

1、严格退订：入住日期前7天取消预订，退还房客50%的房费；如果未能在7天前取消预订，未住宿天数的房费将不予退还；如果提前退房，未住宿天数的房费将不予退还。

2、适中退订：入住日期前5天取消预订，退还房客全部房费；如果未能在5天前取消预订，扣除房客首晚房费，及剩余未住宿天数50%的房费。如果提前退房，扣除房客未住宿的首晚房费，及剩余未住宿天数50%的房费。

3、灵活退订：入住日期前1天取消预订，退还房客全部房费；如果未能在1天前取消预订，扣除房客首晚房费，剩余未住宿天数的房费将全部退还；如果提前退房，扣除房客未住宿的首晚房费，剩余未住宿天数的房费将全部退还。

第三条 违约责任

您在自如民宿上发布的房源状态及房价视为您对自如民宿注册用户做出的有房要约，一旦房客通过自如民宿确认预订即为承诺，即生成有效订单，您应对房客履行提供租赁房屋及服务应尽的义务。

您应及时更新房源状态、租金金额、查收房客预订短信或后台管理系统预订通知并及时回复确认，生成订单且房客成功支付后，您应确保房客的顺利入住以及入住期间的安全，不得擅自变更或取消，在房源信息中有服务承诺的，应履行服务承诺。如您在房客发起预定后12小时内未对订单进行确认或拒绝的，“自如民宿”有权强制取消该订单且有权对您在自如民宿平台上发布的全部或部分房源进行下架处理。

您单方取消订单（包括但不限于直接取消订单、拒绝为房客办理入住等）会影响房客的计划，属于违约行为，所以您需要承担以下违约责任：

1）支付取消违约金，您同意自如民宿从您委托自如民宿代收的最近一笔款项中扣除相应取消违约金，取消违约金按照如下规则计收：6个月内1次取消免收取消违约金，6个月内超过1次，每取消1次收取100元的取消违约金。房客入住前1天房东不允许线上取消订单，如确实需要取消订单需要联系自如民宿客服，房东除承担取消违约金外，还须向自如民宿支付相当于房客已经支付的首晚房费金额的违约金；

2）降低房源排名，自如民宿有权依据您的取消订单率对您的房源排名进行降低；

3）房源下架，自如民宿有权对您在平台上发布的全部或部分房源进行下架处理；

4）自如民宿有权将您的取消订单记录写入芝麻信用及自如信用记录。

如果您认为您的取消行为是由于不可抗力因素导致的（不可抗力因素是指地震、火山、泥石流、战争等您无法预见、无法避免并无法克服的客观情况），则需在取消订单后7日内联系自如民宿客服提出申诉，并按照自如民宿要求提供相关证明，经自如民宿审查确认符合不可抗力情形的，您可免于承担违约责任，自如民宿将退还您已支付的该次取消违约金（如有），重新上架您的房源（如需），撤销您在芝麻信用及自如信用的该次取消记录。

如因房东原因造成房客不能顺利入住的，应向自如民宿承担相当于房客支付的首晚房费金额的违约金，如仍不足以弥补自如民宿损失的，房东仍应承担其他赔偿责任；如因此给“自如民宿”造成其他损失（包括但不限于“自如民宿”自身损失及向房客承担的赔偿责任）的，房东应承担赔偿责任并同意“自如民宿”从代房东收取的费用中扣除赔偿金。 如您与房客协商取消订单的，不视为您单方违约。您可自定义退还给房客的房费，房客确认后，您同意自如民宿将按照协商金额处理订单；或在房客同意的前提下，将房客安排至同区域、同星级或更高级别的酒店、民宿，并承担客户原订单中全部首晚房费及其他相关费用（包括但不限于后入住酒店、民宿与原房费差价、交通费、通讯费等）。 本协议中所称的“房源下架”“下架”是指：该房源所有信息从“自如民宿”上撤销。

第四条 费用与结算

（1）费用

您同意，房客通过平台向您支付的实际消费对应的款项将由自如民宿代您收取。 为了实现房东进入合作平台开展经营活动、进行合同项下交易之目的，自如民宿向您收取服务费用。服务费为租金以及清洁费总和的3％。

（2）结算方式

您可以在自如APP民宿房东端中“我的”-“我的账户”-“结算方式”中选择“按订单结算”或“按天结算”，若您选择“按订单结算”则小于7天的订单每单结算1次；大于7天的订单每7天结算1次。若您选择“按天结算”则每天结算1次。
