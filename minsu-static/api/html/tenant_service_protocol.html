第二条 房源预订

预订房源前，您应仔细阅读房源信息，确保清楚了解房东描述的房源信息、价格、收退款政策等，并接受房东制定的房源使用规则、退订政策等相关约束条件。房源一经预订并支付成功，您与房东即达成交易协议，您应严格遵守相关规则。

房东可将房源设置为申请预订或者立即预定。若为申请预订房源，提交预订后，房东会在规定时间内处理您的订单，确认或拒绝您的预订请求。如房东在规定时间内未处理您的订单，系统则默认此预订订单过期失效；若为立即预定房源，您可直接预订房源，不用等待房东同意即订单成立。

维护平台交易秩序、保护交易双方的安全及隐私，请不要在订单完成支付前向房东提供和索要联系方式，包括但不限于手机/电话号码、电子邮件地址、具体地址、除“自如民宿”外的其它即时通讯工具号码（微信、QQ等）、账号及任何其它联系方式，及获得联系方式的方法。

第三条 费用支付

平台预订页面默认显示房源的日间夜价格，根据您选择的预订住宿日期，系统将会展示该房源在实际住宿日的确切房价。 页面显示价格仅为房源房费，不包含房东自行规定的清洁费、押金以及额外入住人员费用等。自如民宿将按照房费总和的9%向您收取服务费，除您在线支付的全部订单金额及押金外的费用，房东可能将当面向您收取额外费用（如：水电煤气费用等），若因该额外收取费用部分产生任何争议，则自如民宿平台将不予承担任何责任。

提交预订时，页面将提示您房款信息及计算公式，您需仔细核对房费金额，并清楚了解付款规则。

房东确认接受您的预订后，您应在规定时间内，通过“自如民宿”在线成功支付全部订单金额及押金，逾期未成功支付，系统则默认此订单过期失效。

发票：预订房源前您清楚您所预订的房间，房费部分发票由房东提供，您需提前向房东咨询。自如民宿向您提供服务费发票，具体开票金额为实际服务费减去优惠券金额部分。

您与房东自行达成的承诺或协议与“自如民宿”无关，“自如民宿”不对此承诺或协议作出任何解释和承担任何责任。

第四条 取消预订及退款

“自如民宿”上各房源的交易规则由房东自主设定，每个房源详情页和提交预订请求页都清楚列明了交易规则，明确说明取消预订及退款细则，可能依房源不同而规则不同。

“退订政策”：由房东自主设置，若房客于入住日14点的X天（X*24 小时）之前取消预订，则“自如民宿”将向房客全额退还线上支付的全部订单金额及押金费用。

1、严格退订：入住日期前7天取消预订，退还房客50%的房费；如果未能在7天前取消预订，未住宿天数的房费将不予退还；如果提前退房，未住宿天数的房费将不予退还。

2、适中退订：入住日期前5天取消预订，退还房客全部房费；如果未能在5天前取消预订，扣除房客首晚房费，及剩余未住宿天数50%的房费。如果提前退房，扣除房客未住宿的首晚房费，及剩余未住宿天数50%的房费。

3、灵活退订：入住日期前1天取消预订，退还房客全部房费；如果未能在1天前取消预订，扣除房客首晚房费，剩余未住宿天数的房费将全部退还；如果提前退房，扣除房客未住宿的首晚房费，剩余未住宿天数的房费将全部退还。

若您在入住日前取消预订，依订单状态不同，订单金额退还分为如下情况：

尚未完成支付的订单，在取消预订后将不会产生收取任何费用；

已成功支付的订单，若于“退订政策约定日”之前取消，将全额退还您的支付全部房款；

已成功支付的订单，若于“退订政策约定日”之内取消，将按照房东规定的交易规则执行，扣除您部分/全部订单金额后，剩余部分退还给您，但您在支付中使用的用于抵扣部分费用的优惠券等将不予退还；扣除金额将支付给房东作为您的违约赔偿款。 若您在入住后，不能按照订单约定住满入住天数需要提前退房，将按照房东制定的交易规则执行，扣除您部分/全部订单金额后，剩余部分退还给您，但您在支付中使用的用于抵扣部分费用的优惠券等将不予退还；扣除金额将支付给房东作为您的违约赔偿款。

注：如您1.未绑定银行卡支付的，退款（包括押金）将于1个工作日退还至您的自如空间，您可提现至银行卡里，需要3个工作日到账；2.绑定银行卡支付的，退款（包括押金）将于3个工作日退还至您支付的银行卡；3.使用微信支付，退款（包括押金）将于2个工作日原路返回。
